# Geonarrative Landing Page

Static landing page at geonarrative.usgs.gov displaying featured & public geonarratives.

### Development & Testing

Very simple static single page HTML, CSS, and Javascript. No dependencies or build process.

To test locally, start an HTTP server from the project root.

Another easy method is using the [Live Server extension for VS Code](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)


## Deployment

Upload the source files directly to natweb.
